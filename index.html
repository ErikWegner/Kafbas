Kassenabrechnung für Basare

Über

Kafbas entstand zur Unterstützung wohltätiger Basare. Alle Artikel werden mit einer Etikette ausgestattet, die eine Zahlenkombination enthält, dabei enthalten die ersten drei Ziffern die Verkäufernummer und die übrigen Ziffern geben den Preis an.

Nach Beendigung der Verkaufsveranstaltung wird von Kafbas eine Abrechnung für jeden Verkäufer erstellt, einen frei definierbaren Anteil behält der Veranstalter als Provision ein. Die Auswertung enthält dabei sowohl jeden Einzelwert als auch die Gesamtsummen.

Die Kassenarbeitsplätze arbeiten ohne Netzwerk, das Programm sieht die Datenaustauschmöglichkeit vor, dies kann mehrmals erfolgen, muss aber mindestens einmal vor der Abrechnung durchgeführt werden.

Vorteile

  • Programmiert mit Java, dadurch Einsatz unter Microsoft Windows, Mac OS X, Linux, ... möglich
  • Die Eingabe erfolgt vollständig über die Tastatur und kann auf den Ziffernblock beschränkt werden, dadurch schnelle Erfassung mit geringer Fehlbedienungsquote
  • Mehrere Kasseplätze können ohne Netzwerkverbindung genutzt werden, dadurch gibt es weniger Aufwand beim Aufbauen und Betrieb.
  • Detaillierte Auswertung, dadurch schnelle Abrechnung nach dem Ende der Veranstaltung.
  • Sinnvolle Vorbelegung der Parameter und Abfrage der wichtigsten Einstellungen während der Installation, dadurch kann die Anwendung sofort genutzt werden.
  • Ein aussagekräftiges Handbuch beschreibt alle Aspekte des Programms, dadurch können eigene Anforderungen leicht umgesetzt werden.
